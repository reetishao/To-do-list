# To-Do List Manager

A Python command-line task management application that helps users organize tasks, assign due dates, update statuses, and save progress between sessions.

## Features
- View all tasks
- Add new tasks with due dates
- Update task status
- Load tasks from a text file
- Save updates automatically

## Technologies Used
- Python
- Dictionaries
- Functions
- Loops
- Conditionals
- File Handling
